# What OpenAI sends back in every 200 response:
{
  "usage": {
    "prompt_tokens":     15,
    "completion_tokens": 28,
    "total_tokens":      43
  }
}

cost = (prompt_tokens × input_price) + (completion_tokens × output_price)

For gpt-4o-mini:
  input:  $0.15 per 1M tokens
  output: $0.60 per 1M tokens

One typical call (15 in + 28 out):
  = (15 × 0.00000015) + (28 × 0.00000060)
  = $0.000002 + $0.0000168
  = $0.0000188 per call
