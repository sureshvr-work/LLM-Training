RAG at scale — a worker fleet reads, chunks, and embeds

embed-worker-1 loads the PDF and chunks it (4 colours). The fleet sends chunks through NAT to the model replicas. API meters tokens & $; local meters GPU & power.

each worker sends its own colour: worker-1 worker-2 worker-3 worker-4  — square = chunk going out; circle = its vector returning to the same worker, same colour.
